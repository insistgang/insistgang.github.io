感悟

关于 AI 使用历史的回顾：

第一次用这个工具是2024年12月，当时没有铅笔，换了笔触和颜色。用 Cloud 的原因是觉得吸附能力非常强。研一的时候（大概一年半前）它可以通过 SVG 或 HTML 方式画出漂亮的图，用于比赛论文。当时版本是 3.5，现在已经 4.5 了。

参加过上财金融建模比赛、EDA 比赛。虽然没拿奖，但这个工具确实帮了很多忙。朋友刘景用它参加 EDA 比赛，没我这个软件做不出来。被封了很多号，亏了小 1000 块钱。

2-3 月用 ChatGPT 做考研复试辅导，帮本科生做作业，用 200 块一个月的软件挣了 2000 块。4 月低空经济比赛用 ChatGPT 写文案。7 月用 ChatGPT 写网页，体育生网页拿了国三（人工智能创新大赛）。ESG 评分系统、智慧城市系统、双碳系统都没中，但体育生网页中了。

之后又开始用 Claude。用 Claude Code 写了 EDA 竞技挑战赛（3-4 天开发），还有两个国三项目。金融创新项目用了知识蒸馏模型，把 671B 参数的大模型蒸馏成 10B 的小模型。

AI 发展趋势：

  • Manus：智囊团模式，把任务分配给最强的人（Claude 架构、Kimi 前端、GLM 后端）
  • Kimi：笨鸟先飞模式，任务 A 分配给 B1-B10 子任务，总结经验

知识平权：
大部分知识被拉到同一起跑线。只要有知识库就能快速学习。把教资资料放进知识库，Cloud Code 配置学习助手，会安排学习任务、出题、整理资料。

信息差：
能挣钱最多的还是信息差。比如教资资料、考研真题、专业课辅导。

  • 数据集：跑 baseline
  • 苏州：玩（等美赛结束）
