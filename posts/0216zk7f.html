昨日计划完成情况

  • 早睡早起（今日11点起床，部分完成）
  • 继续整理祖师爷论文资源（未完成）
  • 爬虫项目优化（未完成）
  • 运动，恢复锻炼习惯（未完成）

今日记录

上午

11点起床，吃了个午饭。

下午

和发小一起去散步，唠了会嗑。他给我讲了一下他工作的一个商业模式闭环以及接触的老客户把整个运营都做起来了，我觉得也算是超级个体——别人是因为他这个人来买单，而且朋友之间互相介绍他来帮忙做事情。

在家打扫卫生，贴春联。贴完春联之后去打了会牌，一年一度的桌游活动，小赚 20r。

晚上

下午去拿了快递，过年新买的裤子和鞋子到了。和爸妈一起吃了年夜饭，出去买了香和蜡烛。

折腾 GLM 让他帮我深入读论文，今天烧了快 4 亿 token，主要就是 Claude Code 和 OpenCode 两个同时启动开多智能体进行分析。祖师爷的论文还是太难读了。

回来打扫了一下卫生，整理了一下自己的房间。OpenCode 老师出 bug，消息发送不出去等问题。

今年没搞那些活动的红包，还是搞 Kimi 跟 Claude Code 使用。放假的精神也崩的有点紧，感觉没必要这样，虽然 token 在燃烧，但是进展并不快。

简单复习了一些教资的内容，想做的还是很多。

状态

精力 6/10，情绪 6/10

健康打卡

  • 午餐：未记录
  • 晚餐：年夜饭
  • 运动：无
  • 喝水：2L

今日收获

  • 和发小交流，对”超级个体”的商业模式有了新的理解
  • 贴春联、年夜饭，年味渐浓
  • 整理了房间和卫生

今日卡点

  • 祖师爷的论文太难读，进展缓慢
  • OpenCode 出现 bug，影响效率
  • 虽然燃烧了大量 token，但实际进展不快
  • 放假精神绷得太紧

感悟

超级个体的商业模式：别人因为你这个人来买单，朋友间互相介绍带来持续业务。这是个人品牌和信任价值的体现。

假期也要张弛有度，token 在燃烧不代表进展在推进。需要聚焦真正有产出的工作。
