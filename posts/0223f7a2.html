今日记录

上午

8点起床，脑袋昏昏的。吃完饭快9点出去散步。重新编辑了公众号文案，用第三方排版重新排了一下，写了一篇公众号。把昨天做的另一个内容也写成了一篇公众号，做了agent配图。

抖音涨了1000多个粉丝，发了朋友圈，公众号也发了朋友圈作为公域流量入口。

下午

中午给一个兄弟教他如何用open code配置MCP和skills。下午3点在Andy女朋友的电脑上安装好了Claude Code + Kimi Code + OpenClaw。

自己重写了一篇MiniMax使用指南，移动到Hexo博客和公众号上。写了一个skills，将整理好的博客直接给Hexo安排好并做格式标准化。

晚上

更多的是给朋友讲如何将一件事做闭环的逻辑，以及我做出东西的逻辑。给另一个兄弟讲怎么用AI，讲到了12点。

图片转矢量图发论文确实是一个很好的点，值得研究研究。

状态

精力 6/10，情绪 6/10

健康打卡

  • 早餐：青菜牛肉 + 小炒肉丝
  • 午餐：青菜牛肉 + 排骨汤 + 香肠 + 肉圆
  • 晚餐：加了一些毛豆烧丝瓜
  • 运动：散步约 10000 步
  • 喝水：2L

今日收获

  • 跑通了公众号内容发布流程：文案编辑 -> 排版 -> 配图 -> 发布
  • 抖音涨粉1000+，公众号开始公域流量运营
  • 完成MiniMax使用指南
  • 新写了一个Hexo博客发布的skill
  • 帮两个朋友配置AI工具，教学相长

今日卡点

  • 脑袋昏昏，状态一般
  • 没有时间推进论文和红学项目
  • 被教学和分享占用了大量时间

感悟

给朋友讲如何将一件事做闭环的逻辑，教学相长，自己也在梳理思路。
