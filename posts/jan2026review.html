2026年1月复盘

📊 月度概览

1月的核心主题是**”收尾与启动”**——完成了小程序项目的交付，启动了教资备考，搭建了个人知识库，但论文和教资学习推进不足。整体状态前高后低，习惯养成有进展（散步、卸载抖音），但核心目标（论文、教资）持续拖延。

🎯 目标进度

目标 月初计划 实际完成 完成度
论文阅读/写作 每周读3篇 前2周完成，后2周拖了 50%
小程序交付 完成并上线 ✅ 完成交付 100%
软著提交 提交申请 ✅ 已提交 100%
教资备考 启动科目二 科目一启动，科目二未动 30%
数据集采集 完成采集 ✅ 分类整理完成 100%
作息调整 早睡早起 ❌ 持续不规律 0%

🏆 关键里程碑

项目成果

  1. 小程序项目完全交付（1.25）—— bug修复、后台开发、客户满意、代码开源
  2. 软著完成并提交（1.25）—— 源代码、功能说明书、文档材料齐全
  3. 个人知识库搭建（1.25）—— Claude Code + Obsidian，云端同步
  4. 违建数据集采集完成（1.31）—— 7类违建分类整理

个人成长

  1. 人生第10000天（1.16）—— 苏州之行，单纯快乐的一天
  2. 卸载抖音（1.25）—— 时间多了很多
  3. 美赛讲课（1.25）—— 2.5小时培训学弟学妹
  4. 论文打卡连续7天（1.5-1.11）—— ResNet→PVT，CV骨干网络完整梳理

📈 趋势分析

精力/情绪月度走势

观察：精力呈下降趋势，前两周活动多状态好，后两周因拖延和导师事件状态下滑。

习惯执行率统计

习惯 第2周 第3周 第4周 第5周 月度总计
论文阅读 7/7 5/7 2/7 0/7 14/28 (50%)
散步/运动 7/7 7/7 4/7 1/7 19/28 (68%)
喝水2L 6/7 7/7 4/7 6/7 23/28 (82%)
教资学习 - - 2/7 0/7 2/14 (14%)

观察：

  • 散步习惯养成较好（68%），前两周全勤
  • 喝水习惯稳定（82%）
  • 论文阅读前两周好，后两周崩了
  • 教资学习几乎没动

时间分配（粗略估计）

  • 项目开发（小程序/软著）：30%
  • AI工具折腾（Claude Code/manus/kimi）：20%
  • 数据采集：15%
  • 论文阅读：10%
  • 社交/活动：15%
  • 教资学习：5%
  • 其他（整理/研究工具）：5%

🔍 问题诊断

反复出现在周复盘里的问题

问题 出现周数 根因分析
作息不规律 4周 缺乏硬性约束，”说得多做得少”
论文拖延 3周 从”推进”变成”准备”，始终不启动
教资学习不动 2周 优先级被挤掉，没有固定时间块
多线程问题 3周 想要的太多，注意力分散

核心问题：执行力 vs 意愿

模式识别：

  • ✅ 收尾类任务执行力强（小程序、软著）
  • ✅ 新工具学习热情高（Claude Code、manus、kimi）
  • ❌ 深度工作持续困难（论文、教资）
  • ❌ 习惯约束力弱（作息、运动）

深层原因：

  1. 多巴胺陷阱——新工具带来即时反馈，论文学习反馈延迟
  2. 完美主义——总觉得”还没准备好”，不敢开始
  3. 缺乏外部约束——论文、教资没有deadline，容易被挤掉
  4. 精力管理不当——作息不规律导致精力持续下降

🌱 成长记录

技术能力

  • AI工具掌握：Claude Code、Obsidian知识库、manus、kimi agent集群
  • 开发能力：小程序开发（从0到交付）、软著申请流程
  • 学术积累：CV骨干网络（ResNet→PVT）、DETR系列、小目标检测

认知升级

  1. “专注和聚焦，不要想着多开” —— 第2周感悟，但后续未执行到位
  2. “人与人之间的差距在于执行力” —— 第4周感悟
  3. “痛苦的事情才能让我成长” —— 第5周感悟
  4. “信息差是最赚钱的” —— 第5周AI使用感悟

习惯养成

  • ✅ 卸载抖音（持续）
  • ✅ 散步/运动习惯（68%执行率）
  • ✅ 每日复盘习惯（坚持整月）
  • ❌ 早睡早起（0%）

📅 2月规划

要继续保持的

  1. 知识库复盘 —— 每日/周/月复盘体系已建立，继续保持
  2. 散步习惯 —— 效果明显，恢复到68%水平
  3. AI工具学习 —— 但要控制时间，不吞噬正事时间
  4. 卸载抖音 —— 时间充裕，坚持住

要调整改进的

  1. 论文baseline —— 2月必须跑起来，不要再准备了
  2. 教资学习 —— 每天30分钟固定时间块，不再找借口
  3. 作息调整 —— 23点前睡觉，硬性约束
  4. 单线程优先 —— 一件事做完再做下一件

2月核心目标（不超过3个）

  1. 论文baseline跑出结果
  2. 教资科目一/二各推进50%
  3. 作息恢复到23点前睡

💬 一句话总结

1月是”收尾与启动”的月份——项目收尾做得好，深度学习启动失败，2月要少折腾工具，多做难事。
