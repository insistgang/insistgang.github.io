第四部分：总结与建议

Francis当前最大的时间黑洞

“帮人做事”是最大的时间黑洞

典型案例：

  • 3/17 下午帮纪元安装 Codex + Claude Code + Codex，技能安装 → 占用2小时
  • 3/21 下午帮纪元女朋友安装 CLI → 占用2小时
  • 3/25 下午帮雅元和程果配置Claude Code、帮思宇点赞下载软件 → 碎片化

帮人做事本身不是问题，问题在于：

  1. 黄金时段被占用（下午本应是论文时间）
  2. 没有边界感（不好意思拒绝）
  3. 帮完后没有记录时间价值（不知道投入产出比）

论文投稿前最需要改变的一个习惯

建立”论文优先”的自动化触发机制

当前问题：论文阅读计划从5分钟降到1分钟仍无法坚持，说明问题不在难度，而在触发机制缺失。

建议方案：

  1. 物理触发：早上到实验室第一件事就是打开论文PDF（不需要读，只需要打开）
  2. 习惯叠加：把”论文阅读”叠加到已有的习惯上，如”健身后/吃饭后/到实验室后”
  3. 降低门槛到零：今天的论文任务 = 打开PDF，仅此而已
