2026年2月24日 星期一

昨日计划完成情况

  • 早睡早起（23:30前睡，8:30前起）- 8点醒，达标
  • 读论文（每日KPI）- 整理了claude 7大功能
  • 推进红学论文
  • 研究图片转矢量图发论文的可行性 - 有研究

今日记录

上午

8点钟醒，9点吃早饭（大杂烩、牛肉、肉圆、香肠）。先把昨天发的minimax agent公众号发掉了，用minimax agent给一篇notion作者的文章配了图并整理后发了公众号。10点钟睡到了11点。

下午

去舅奶奶家吃午饭，和亲戚唠嗑，吃了很多菜。

研究了superpowers、planning with files、notebookLM这三个claude code skills，都下载到本地了。用superpowers查看之前实习的项目，用planning with files整合博客仓库并对博客进行升级，升级过程中产生了很多问题，不断修复。

看到Qwen的coding plan模型在openclaw效果很好，以及一站式博文分发平台，加上社交平台的激励计划，能抓住这波浪潮的机会。

给minimax agent专门写了一个我需求的专家提示词，总结之前的提示词。把最新博文放到claude code里发现skills没有执行，才发现自己写的skills需要加到memory里面才可以用。

考研成绩下午五点还没出来。智谱GLM一直连不上。研究了AI绘图的矢量图。

晚上

又是舅奶奶家吃晚饭，吃的很多。

姐姐说网上很多人推学英语的网站俱乐部.co，发现agent for education是风口。大家开始转副业找自己能做的点，做自媒体成本越来越低。

看到EvoMap创造了AI Agent进化协议，千问模型在OpenClaw效果好，DeepSeek接OpenClaw做量化比较好（DeepSeek本身就是做量化出身的）。

Kiro免费积分还剩一天过期，约400多积分，计划用掉重构之前的项目。用claude整理了claude的7大功能，写成了公众号。

状态

精力 6/10，情绪 6/10

健康打卡

  • 早餐：大杂烩、牛肉、肉圆、香肠
  • 午餐：舅奶奶家，很多菜
  • 晚餐：舅奶奶家，很多菜
  • 运动：未记录
  • 喝水：2L

今日收获

  • 研究了superpowers、planning with files、notebookLM三个claude code skills
  • 升级了博客
  • 写了minimax agent专家提示词
  • 整理了claude的7大功能写成公众号
  • 了解到agent for education是风口

今日卡点

  • 智谱GLM一直连不上
  • skills需要加到memory才能用
  • 时间精力花在有趣的事情上，而非应该做的论文

感悟

痛点在于把时间和精力花在有趣的事情（大语言模型研究）上，而非应该做的论文难题上。Agent for Education是风口，自媒体成本越来越低。

明日计划

  • 早睡早起（23:30前睡，8:30前起）
  • 读论文（每日KPI）
  • 推进红学论文
  • 用掉Kiro积分重构项目
  • 图片转矢量图研究
