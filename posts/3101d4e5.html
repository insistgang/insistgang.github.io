  • 数据集分类整理完成，共7类违建

  • kimi2.5 的 agent 集群真的厉害
  • 美赛 A 题、B 题 都挺有意思

今日卡点

  • 还没跑 baseline
  • Claude Code 配置复杂，把自己搞炸了
  • clawdbot 修复后图片打不开等问题
  • 很多事情觉得没有意义，永远解决不完

感悟

有钱人的孩子大多不会娇生惯养，他们会经历更多磨炼。我和很多厉害的人交流，他们讲话都很慢，反倒是我讲话非常快，做任何事都急赶着急的，非常的不对，得好好反思一下。

现在各种大模型生成的内容你不懂原理几乎都是黑盒。

明日计划

  • 美赛：继续研究
  • 数据集：跑 baseline
  • 苏州：玩（等美赛结束）
